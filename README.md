🌲 Forest Frontier – Jeu de plateforme 2D
Description

Forest Frontier est un jeu platformer 2D, développé en Python, dans lequel le joueur doit accomplir et réussir quatre niveaux liés à la protection de la nature et à la préservation de l’environnement. Chaque niveau introduit de nouvelles mécaniques de jeu et des défis progressifs

🎮 Lancement du jeu
Pour jouer :

Ouvrez le projet dans un IDE Python (ex. VS Code, PyCharm, Thonny).

Exécutez le fichier main.py pour démarrer le jeu.

🕹️ Description des niveaux

Niveau 1 – Plantation :
Le joueur doit ramasser les graines données par les oiseaux, puis les planter et les arroser pour les faire pousser. Ce niveau n’a pas de limite de temps et permet de découvrir les premières mécaniques du jeu.
Niveau 2 – Incendie dans la forêt :
Le joueur doit éteindre les flammes qui apparaissent dans la forêt avant la fin du temps imparti. S’il met trop de temps ou laisse trop de feux actifs, il perd le niveau.
Niveau 3 – Tri des déchets :
De nombreux déchets apparaissent autour du joueur. Il doit les ramasser et les déposer dans la poubelle correspondant à leur type : verre, plastique ou restes.
Niveau 4 – Combat final :
Le joueur affronte le boss final, qui tire des projectiles vers lui. Le joueur doit les éviter pour ne pas perdre de points de vie. Parfois, le boss fait tomber une arme que le joueur peut ramasser et utiliser pour le vaincre.

💡 Points clés

Jeu codé intégralement en Python.

Mécaniques de plateforme, gestion d’objets et progression de difficulté.

Présence d’un menu principal, d’un bouton paramètres et d’un système de niveaux.

Thème écologique mettant en avant les gestes responsables et la préservation de la forêt.
